# Design Feedback for Steady Thread Media Website

## 1. Sections Should Flow Instead of Stack

Right now the page feels like **a series of square blocks stacked vertically**. I would like the sections to feel more connected and fluid.

Suggested improvements:

* Introduce **curved section dividers**
* Use **subtle wave or arc transitions between sections**
* Avoid hard straight edges between every section
* Some sections can slightly **overlap or bleed into each other**

Example ideas:

• Curved divider between hero and next section
• Diagonal or rounded section backgrounds
• Large soft shapes behind content blocks

The goal is to create **visual flow down the page** instead of rigid rectangles.
