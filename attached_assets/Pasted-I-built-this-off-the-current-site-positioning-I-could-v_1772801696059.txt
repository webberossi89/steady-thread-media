# 2. Replit build structure

Use this page/component structure:

```txt
/
  index.html
  /services/index.html
  /approach/index.html
  /case-studies/index.html
  /insights/index.html
  /about/index.html
  /contact/index.html
  /blog/[slug].html or route-based if using a framework

/assets
  /css
    base.css
    layout.css
    components.css
    pages.css
  /js
    main.js
  /img
    hero/
    icons/
    case-studies/
```

If you are using a modern Replit stack, I would build this in **Next.js or Astro**.
My preference for this site is:

* **Astro** if mostly content/marketing site
* **Next.js** if you want easier future CMS integration and dynamic case studies/blog

For speed and clean performance, Astro is the better fit.

# 3. Design system for the site

Using the brand guide direction:

## Colors

* Primary background: `#1A1A1A`
* Accent 1: `#F0D6FF`
* Accent 2: `#FFFFEB`
* White: `#FFFFFF`
* Soft neutral text on dark: `#D9D4CF`

## Typography

* Heading font: editorial serif style
* Body font: Inter or similar sans-serif
* H1 large and high contrast
* generous whitespace
* avoid dense paragraphs

## Visual direction

* dark hero sections
* cream or soft-light content sections
* lavender used selectively for emphasis
* pill-style tags/buttons for topic labels and micro-navigation
