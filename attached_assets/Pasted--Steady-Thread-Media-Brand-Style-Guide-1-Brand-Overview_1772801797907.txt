# Steady Thread Media

Brand Style Guide

---

# 1. Brand Overview

Steady Thread Media is positioned as a **modern B2B growth consultancy** focused on helping companies build **structured, predictable go-to-market systems**.

The visual brand reflects:

* clarity
* precision
* editorial sophistication
* modern SaaS aesthetics

The overall look should feel **minimal, strategic, and premium**, avoiding loud or overly playful design.

# 3. Color Usage Ratio

Recommended usage across the site:

| Color              | Usage  |
| ------------------ | ------ |
| Charcoal (#1A1A1A) | 65–70% |
| Cream (#FFFFEB)    | 20–25% |
| Lavender (#F0D6FF) | 5–10%  |

This maintains a **minimal and premium feel**.

# 4. Typography

## Heading Font

Style category:

**Modern Editorial Serif**

Recommended font options:

| Font             | Type             |
| ---------------- | ---------------- |
| Canela           | Premium          |
| Recoleta         | Premium          |
| Editorial New    | Premium          |
| Playfair Display | Free alternative |

Recommended implementation:

Primary heading font
**Playfair Display** (for simplicity and accessibility)

---

## Body Font

Recommended font:

**Inter**

Reason:

* extremely readable
* modern SaaS standard
* pairs well with serif headlines

---

## Typography Hierarchy

| Element    | Font  | Weight    | Size    |
| ---------- | ----- | --------- | ------- |
| H1         | Serif | Bold      | 56–72px |
| H2         | Serif | Semi Bold | 40–48px |
| H3         | Serif | Medium    | 28–32px |
| Body       | Inter | Regular   | 16–18px |
| Small text | Inter | Regular   | 14px    |

# 10. Imagery Style

Use imagery that is:

* minimal
* soft lighting
* neutral color palette
* professional and modern

Avoid:

* overly bright stock photos
* heavy color saturation
* playful or cartoonish graphics

---

# 11. Icon Style

Icons should be:

* thin line icons
* minimal
* monochrome
* lavender for active states

---

# 12. Brand Voice

Tone should feel:

* clear
* confident
* thoughtful
* strategic

Avoid:

* buzzword heavy language
* hype marketing
* exaggerated claims

Messaging should emphasize:

* structure
* clarity
* strategic thinking
* sustainable growth

---

# 13. Website Style Summary

The Steady Thread visual identity should feel like:

* modern SaaS platform
* editorial consulting firm
* minimalist design system

Comparable design styles:

* Linear
* Vercel
* Notion
* Superhuman

---

# 14. Logo Usage

The logo should appear:

* white on dark backgrounds
* charcoal on light backgrounds

Minimum spacing:

```
0.5x logo height on all sides
```

Avoid:

* stretching
* adding drop shadows
* recoloring the logo.

---

# 15. Core Brand Principles

Design decisions should follow these principles:

1. Clarity over decoration
2. Minimal over complex
3. Strategic over trendy
4. Calm visual hierarchy
5. High readability
